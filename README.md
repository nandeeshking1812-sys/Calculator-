# Advanced Calculator

A scientific calculator with memory functions, history, and custom variables.

## Features
- Basic math (+, -, ×, ÷, %, ^)
- Scientific functions (sin, cos, tan, log, sqrt, factorial)
- Memory (M+, M−, MR, MC)
- History tracking
- Custom variables
- Keyboard support

## How to Use
1. Open `index.html` in any browser
2. Click buttons or use keyboard
3. Press = or Enter for result
